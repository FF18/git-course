При работе в команде важно видеть "подписи" тех с кем вы работаете . Для этого есть команда config.
Настройка значений конфигурации Git на глобальном и локальном уровнях проекта.
git config --global user.name = "Felix" - ("Etel")
git config --global user.email = "" 

git init - в пустой  директории устанавливает папку .git - инициализируем наш repository
git add <file.format> - (for example index.html) - добавить один файл в git
git status - показывает что сейчас происходит в git
git rm --cached <file.format> - git перестаём следить за данным файлом 
cls - очищает terminal
git add . - добавляет все файлы в git и измененные  тоже 

git commit --message "first commit" или git commit -m "first commit"
